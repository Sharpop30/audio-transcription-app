Audio Transcription App

Convert your audio files to text using OpenAI's advanced transcription

How to Get Started with OpenAI API:

  1. Go to OpenAI's platform and create an account
  2. Navigate to the API section in your account
  3. Click on "Create new secret key"
  4. Copy your API key and paste it in the field below
  5. Your API key will be saved locally and used only for transcription
